cUser Stories

These user stories are derived from the Airbnb Clone use case diagram and capture the core interactions between the system and its users.
 1. User Registration
As a user, I want to be able to register an account so that I can book properties or list my own.
2. User Login
As a user, I want to log in securely using my email and password so that I can access my dashboard and manage my bookings or listings.
3. Property Listings
As a host, I want to create, edit, and delete property listings so that I can make my properties available to potential guests.
4. Booking a Property
As a guest, I want to search for available properties and make bookings based on my preferred location, dates, and budget.
5. Payment Processing
As a user, I want to make secure payments for my bookings so that I can complete the reservation process.
 6. Admin Management
As an admin, I want to be able to manage users, listings, and bookings so that I can ensure smooth operation of the platform.
